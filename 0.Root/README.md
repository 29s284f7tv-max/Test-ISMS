# ISMS – test-Firma

Willkommen im Informationssicherheits-Managementsystem (ISMS) der **test-Firma**.

Dieses Repository enthält:
- die Leitlinie der Informationssicherheit
- Policies (Regelwerke)
- Prozesse
- Risikoanalyse & Asset-Management
- Maßnahmen (Controls)
- Nachweise (Evidence)
- Kennzahlen (KPIs)
- Audits & Reviews

Das ISMS folgt einem Minimalansatz, der dennoch alle Anforderungen an ein wirksames und überprüfbares System erfüllt.

## Struktur
- `/Leitlinie.md`
- `/scope.md`
- `/roles.md`
- `/policies/`
- `/procedures/`
- `/risk/`
- `/controls/`
- `/evidence/`
- `/metrics/`
- `/audit/`
- `/awareness/`

## Versionierung
Alle Änderungen werden über Git-Commits dokumentiert, um Revisionssicherheit und Nachvollziehbarkeit sicherzustellen.
